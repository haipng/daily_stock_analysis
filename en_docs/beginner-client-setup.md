# Beginner Client Installation and Configuration Guide

This guide is written for users with no coding experience who just want to download the client and use it directly. The goal is simple: download the client, fill in a model service key (Key), enter stock codes, then generate your first analysis report.

> The reports generated by this project are supplementary analysis only, not investment advice. Please make your own risk judgments for real trading.

## Prerequisites

1. Windows or macOS computer.
2. A model service key (Key), recommended to choose one from below:
   - [Anspire Open](https://open.anspire.cn/?share_code=QFBC0FYC): Supports mainstream global models; one Key can be used for both models and news search, easiest for first-time setup.
   - [AIHubMix](https://aihubmix.com/?aff=CfMq): Supports mainstream global models; suitable for users wanting to switch between multiple models on one platform.
3. Stock codes you want to analyze, e.g., `600519,hk00700,AAPL`.

## 1. Download Client

Open release page:

<https://github.com/ZhuLinsen/daily_stock_analysis/releases/latest>

Download from `Assets` (attachments) at page bottom:

| Computer | Download This |
| --- | --- |
| Windows | `daily-stock-analysis-windows-installer-<version>.exe` |
| Windows don't want install | `daily-stock-analysis-windows-noinstall-<version>.zip` |
| macOS Apple Silicon | `daily-stock-analysis-macos-arm64-<version>.dmg` |
| macOS Intel | `daily-stock-analysis-macos-x64-<version>.dmg` |

Don't download `latest.yml`, `*.blockmap`, they are not client installers.

Don't know which Mac chip: Click Apple icon top-left → About This Mac; if you see M1/M2/M3/M4 choose `arm64`, if you see Intel choose `x64`.

## 2. Install and Open

- Windows installer: Double-click `.exe`, follow prompts, default install directory is fine.
- Windows portable: Extract `.zip`, double-click `Daily Stock Analysis.exe`.
- macOS: Double-click `.dmg`, drag app to `Applications`. If it says "unverified developer," allow it in System Settings → Privacy & Security.

macOS users upgrading should export configuration backup first in client settings.

## 3. Configure AI Model

Open client, go to:

`System Settings → AI Models`

Choose just one plan below.

> Important: After changing any settings, click the save button on page; after seeing save success notification, switch pages or go back to home.

### Plan A: Anspire Open

1. Open [Anspire Open](https://open.anspire.cn/?share_code=QFBC0FYC), register/login then create API Key.
2. Back to client, in Quick Add Channel select `Anspire Open`.
3. Paste API Key.
4. Model name: select already-enabled model from console; if unsure, choose console-recommended or lightweight model.
5. Click save; after seeing success, click test connection.

### Plan B: AIHubMix

1. Open [AIHubMix](https://aihubmix.com/?aff=CfMq), register/login then create API Key.
2. Back to client, in Quick Add Channel select `AIHubmix (Aggregated Platform)`.
3. Paste API Key.
4. Model name: select already-enabled model from console; if unsure, choose console-recommended model.
5. Click save; after seeing success, click test connection.

See test success, then continue next step.

## 4. Fill Watchlist

Go to:

`System Settings → Basic Settings`

Find `Watchlist`, enter:

`600519,hk00700,AAPL`

Multiple stocks separated by English commas. Common formats:

- A-shares: `600519`, `300750`, `000001`
- Hong Kong: `hk00700`, `hk09988`
- US stocks: `AAPL`, `TSLA`, `NVDA`

After filling, click save, see success then back to home.

## 5. Recommended: Configure News Source

News source is not required, but recommended. It affects recent news, announcements, event drivers, market trends, and risk warnings.

Go to:

`System Settings → Data Sources`

Choose based on your model service:

1. Using Anspire Open: Find `Anspire API Keys`, fill in the same Anspire Key, save success then done.
2. Using AIHubMix: Recommend also applying for [SerpAPI](https://serpapi.com/baidu-search-api?utm_source=github_daily_stock_analysis) or [Tavily](https://tavily.com/) Key, fill into `SerpAPI API Keys` or `Tavily API Keys`, save success then done.

Can skip news source for trial too; client still generates basic analysis.

## 6. Start Analysis

Back to home:

1. Enter stock code, e.g., `600519`.
2. Click analyze.
3. Wait for task to change from queued → analyzing → complete.
4. View report in history.

## Common Questions

### Download page has many files, which one?

Regular Windows users download `.exe` installer. Don't download `latest.yml` or `*.blockmap`.

### Filled API Key still doesn't work?

Check these:

1. Did you copy complete Key without extra spaces?
2. Does platform account have balance or quota?
3. Is current model enabled?
4. Does test connection show model not found, permission denied, or insufficient balance?

### Configuration messed up?

Export configuration backup in client settings. If problems occur, can import earlier backup, or just reconfigure the three main items: AI Model, Watchlist, News Source.
